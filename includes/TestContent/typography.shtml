Wrap headers in '< header >' tags as this will allow for sub-header/supportive text to be added to the header. Sub-header text can have a different appearence than main paragraph text.

Readability

Make sure that text has enough contrast between itself and it's background. Use one of the tools in the 'Tools' section of this document.

Paragraph

Do not allow line lengths to exceed 75 characters.

Lists

List items start with lowercase letters and never end with a full stop.
